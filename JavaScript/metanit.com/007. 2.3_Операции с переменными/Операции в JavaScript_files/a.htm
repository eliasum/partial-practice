                  !=
                  Сравнивает два значения, и если они не равны, возвращает true, иначе возвращает false: x != 5

                  !==
                  Сравнивает два значения, и если они не равны, возвращает true, иначе возвращает false: x !== 5

                  если два оператора ведут себя одинаково, назревает логический вопрос - зачем один из них. или же автор просто не соизволил объяснить их значение подробно

                        // постфиксный инкремент
                        var a = 5;
                        var b = a++;
                        console.log(a); // 6
                        console.log(b); // 5
                        Постфиксный инкремент аналогичен операции:
                        a = a + 1; // a++

                        тогда заменим на a++
                        var a = 5;
                        var b = a = a + 1;;
                        console.log(a); // 6
                        console.log(b); // 6 не пять!
                        Получается префиксный икремент аналогичен,а не постфиксный.
