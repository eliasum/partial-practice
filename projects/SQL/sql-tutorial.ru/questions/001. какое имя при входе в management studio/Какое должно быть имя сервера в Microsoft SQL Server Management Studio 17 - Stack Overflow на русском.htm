Скорее всего вы не производили установку SQL Server, а поставили только SQL Server Management Studio.

Если вам нужен SQL Server - для ознакомления предлагаю использовать версию SQL Server Express 2016 после установки подключение можно произвести к серверу по следующему имени: .\SQLExpress
