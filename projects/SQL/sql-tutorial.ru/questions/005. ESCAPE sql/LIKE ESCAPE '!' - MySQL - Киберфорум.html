    Решение

    А вы документацию хорошо читали ?

    Символы '_' и '%' это подстановочные символы.
    Первый означает - "любой один символ" , второй 'любое количество любых символов'

    Escape символ - это символ, который позволяет экранировать подстановочные символы '_' и '%', чтобы они не воспринимались как подстановочные.

    Чтобы экранировать подстановочный символ, вы должны были ПЕРЕД ним написать Escape символ:

    SQLВыделить код
    1

      SELECT * FROM  my_table WHERE  col LIKE  'text!%'   ESCAPE '!'
    Вот в таком виде процент не будет восприниматься как подстановочный, а будет восприниматься как просто процент.
    Кстати, по умолчанию, если специально не указывать Escape символ - он равен '\' - косой черте.

    Если вы указали Escape символ, но не использовали его в строке поиска - то это вообще не имеет смысла.
