                                                                                                                                                  Здравствуйте. После просмотра кода класса Bank<t> задумался о необходимости использования ограничения на базовый класс.

                                                                                                                                                  По сути, данное ограничение необходимо для гарантирования присутствия определённых членов класса (как правило, методов) в параметре типа, чтоб компилятор точно знал, что параметр типа гарантированно обладает определёнными методами, свойствами и мог их использовать.

                                                                                                                                                  Но, поскольку по ссылке на базовый класс можно обращаться к классам наследникам, то вместо использования параметра типа с ограничением на базовый класс, можно ввести в класс ссылку на базовый класс, которая аналогично гарантирует наличие определённых членов класса и позволяет получить доступ как к экземпляру своего класса, так и к любому его наследнику.

                                                                                                                                                  Единственное, нужно, чтоб в классах наследниках используемые методы были переопределены, если это необходимо.

                                                                                                                                                  То есть, вместо обобщений с ограничением на базовый класс можно использовать полиморфизм.

                                                                                                                                                  Таким образом, класс Bank<t> можно заменить следующим:

                                                                                                                                                  class Bank
                                                                                                                                                  {
                                                                                                                                                  Account[] accounts;

                                                                                                                                                  public Bank(Account[] accs)
                                                                                                                                                  {
                                                                                                                                                  this.accounts = accs;
                                                                                                                                                  }

                                                                                                                                                  public void AccountsInfo()
                                                                                                                                                  {
                                                                                                                                                  foreach(Account acc in accounts)
                                                                                                                                                  {
                                                                                                                                                  Console.WriteLine(acc.Id);
                                                                                                                                                  }
                                                                                                                                                  }

                                                                                                                                                  }

                                                                                                                                                  Правда, ограничение на базовый класс можно использовать одновременно с другими ограничениями, такими как ограничение на интерфейс или конструктор, что предъявляет больше требований к аргументу нежели ссылка на базовый класс.

                                                                                                                                                  В итоге, ограничение на базовый класс действительно решает какие-то задачи или это всего лишь синтаксический сахар?
