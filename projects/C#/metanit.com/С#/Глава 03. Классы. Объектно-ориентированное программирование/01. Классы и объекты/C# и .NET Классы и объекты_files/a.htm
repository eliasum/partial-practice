            Если первый ссылается на второй, а второй на третий - это не означает, что первый ссылается на третий.
            Если Вы создадите объект класса Person без параметров [new Person();] - будет вызван первый конструктор, который в свою очередь вызовет второй с параметром "Неизвестно", то есть name будет "Неизвестно", a age присвоится значение по умолчанию - 18. // Name - "Неизвестно" Age - 18.
            Если создать объект с одним параметром [new Person("Tom");], то будет вызван второй конструктор, который вызовет третий с параметром "Tom" и значением age 18. // Name - "Tom" Age - 18.
            И если указать оба параметра, то будет сразу вызван третий, где все просто - переданный параметр name присвоится полю объекта, age - соответсвенно так же. // Name - "Tom" Age - 25.
