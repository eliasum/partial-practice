                Парочку комментариев, исходя из собственных экспериментов:
                1. Capacity автоматически увеличивается при добавлении элементов на значение, равное степени двойки. То есть при добавлении трех элементов Capacity = 4, при добавлении пяти элементов Capacity = 8. Но мы можем вручную изменить это значение.
