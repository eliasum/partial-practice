                                                                                                                  Чтобы форыч смог перебрать массив, в классе этого массива, должно быть реализовано два интерфейса IEnumerable и IEnumerator.
                                                                                                                  Форыч вызывает метод GetEnumerator(), который возвращает объект типа IEnumerator.
                                                                                                                  У полученного объекта, в цикле while, форыч вызывает метод MoveNext(), затем получает данные с помощью свойства Current, while выполняется пока MoveNext() возвращает true.

                                                                                                                  Проверил работает:

                                                                                                                  int[] Number = new int[3] { 1, 2, 3 };

                                                                                                                  IEnumerator Enum = Number.GetEnumerator();
                                                                                                                  while(Enum.MoveNext())
                                                                                                                  {
                                                                                                                  Console.WriteLine(Enum.Current);
                                                                                                                  }
