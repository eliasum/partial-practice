                                                                                                      вычислим 4! это произведение чисел 1*2*3*4

                                                                                                      4! = 3!*4;
                                                                                                      3! = 2!*3

                                                                                                      1! = 0!*1
                                                                                                      0! = 1

                                                                                                      1! = 1*1=1

                                                                                                      3! = 2*3=6
