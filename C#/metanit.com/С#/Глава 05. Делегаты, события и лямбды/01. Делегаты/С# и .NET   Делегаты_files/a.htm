                                                                                                                                                              получается что компилятор не может изменить значение переменной , если этой переменной не было еще присвоено никакое значение?

                                                                                                                                                              Может изменить, но не может использовать значение переменной.

                                                                                                                                                              Переменная operation является локальной по отношению к методу. Такие переменные должны быть явно инициализированы в коде до появления любого оператора, в котором используются их значения.

                                                                                                                                                              В вашем коде явная инициализация может произойти внутри логической конструкции

                                                                                                                                                              if (userChoise == 'a')
                                                                                                                                                              {
                                                                                                                                                              operation = Add;
                                                                                                                                                              }
                                                                                                                                                              else if (userChoise == 'm')
                                                                                                                                                              {
                                                                                                                                                              operation = Multiply;
                                                                                                                                                              }

                                                                                                                                                              Но она не произойдёт в случае

                                                                                                                                                              userChoise != 'a' && userChoise != 'm'

                                                                                                                                                              Компилятор сразу видит, что явная инициализация некорректная, что существуют варианты, при которых переменная не будет инициализирована явно, поэтому запрещает далее использовать значение переменной при её вызове в коде operation(a, b);.

                                                                                                                                                              А вот если предусмотреть такой вариант и полностью реализовать явную инициализацию, например, путём

                                                                                                                                                              if (userChoise == 'a')
                                                                                                                                                              {
                                                                                                                                                              operation = Add;
                                                                                                                                                              }
                                                                                                                                                              else if (userChoise == 'm')
                                                                                                                                                              {
                                                                                                                                                              operation = Multiply;
                                                                                                                                                              }
                                                                                                                                                              else
                                                                                                                                                              {
                                                                                                                                                              operation = null;
                                                                                                                                                              }

                                                                                                                                                              либо как вы обсуждали выше путём инициализации сразу при объявлении переменной operation, то код компилироваться будет.

                                                                                                                                                              Программа даже будет работать, когда для userChoise введено 'a' или 'm', но в остальных случаях переменная operation будет инициализирована значением null, поэтому вызов делегата operation(a, b); бросит исключение, о чем автор статьи уже рассказал в пункте "Вызов делегата".
